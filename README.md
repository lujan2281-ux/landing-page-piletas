# Landing Page - Servicios de Piletas

Este proyecto es una página web responsiva desarrollada para un negocio real ("Daniel Lezcano"), orientada a ofrecer servicios profesionales de iluminación, mantenimiento y climatización de piletas.

##  Tecnologías utilizadas
* HTML5, CSS3, JavaScript
* Hosting y despliegue continuo mediante Netlify

## Objetivo del Proyecto
El sitio fue diseñado como una *landing page* comercial, optimizada para presentar servicios técnicos de forma clara y directa. 
Este proyecto demuestra la capacidad para gestionar el ciclo de vida completo de un desarrollo web front-end: desde la maquetación inicial hasta su publicación y puesta en producción en un servidor real, resolviendo una necesidad comercial concreta.

 **Ver proyecto en vivo:** [https://daniellezcano-piletas.netlify.app/](https://daniellezcano-piletas.netlify.app/)
